Acadian--Shortest Path Finding Game By Sanchit

Steps to Play: -

  • Please select the 'Start & End' location by clicking on any cell.
  • Drop down will open with three options - 'Start' , 'End' , 'Block', select option accordingly.
  • Once done, please click on 'Find Path' button. Shortest path display in 'Green' color.
  • Error message will appear if 'No Path' available.
  • Click on 'Reset' button to play again--Enjoy!!

    • **To accomplish this task I have used Breadth first approach(BFS) with Queue structure..
